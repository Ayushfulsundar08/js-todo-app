# Sleek To-Do Web App

This is a modern, responsive To-Do list app built using HTML, CSS, and JavaScript.

### 🔥 Features
- Clean, modern UI with dark mode 🌙
- Add tasks and mark them complete
- Fully responsive design
- Smooth theme toggle

### 🧠 Technologies Used
- HTML5
- CSS3 (Flexbox + Variables)
- JavaScript (Vanilla)
